📊 GOOGLE ADSENSE SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: GET YOUR ADSENSE CODE
──────────────────────────────

1. Login to: https://www.google.com/adsense/

2. Click: "Ads" → "By ad unit"

3. Create new ad unit:
   - Display ads (Recommended)
   - In-feed ads
   - In-article ads
   - Multiplex ads

4. Copy the COMPLETE ad code

5. It will look like:

   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
        crossorigin="anonymous"></script>
   <ins class="adsbygoogle"
        style="display:block"
        data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
        data-ad-slot="XXXXXXXXXX"
        data-ad-format="auto"
        data-full-width-responsive="true"></ins>
   <script>
        (adsbygoogle = window.adsbygoogle || []).push({});
   </script>


STEP 2: ADD CODE TO WEBSITE
────────────────────────────

1. Open: settings.html

2. Scroll to: "Google AdSense Configuration"

3. Paste your code in the text area

4. Click: "💾 Save AdSense Code"

5. Done! Ads will now show on all pages


WHERE ADS WILL APPEAR:
──────────────────────

✅ Homepage (index.html) - 2 ad slots
✅ News page (news.html) - 2 ad slots
✅ Mall page (mall.html) - 2 ad slots
✅ All generated post pages - 1 ad slot each


📈 GOOGLE ANALYTICS SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: GET YOUR MEASUREMENT ID
────────────────────────────────

1. Login to: https://analytics.google.com/

2. Click: Admin (gear icon)

3. Select: Property → Data Streams

4. Click: Your web stream

5. Copy: "Measurement ID"
   Format: G-XXXXXXXXXX (GA4)
   Or: UA-XXXXXXXXX-X (Universal Analytics)


STEP 2: ADD TO WEBSITE
───────────────────────

1. Open: settings.html

2. Scroll to: "Google Analytics Configuration"

3. Paste your Measurement ID

4. Click: "💾 Save Analytics ID"

5. Done! Tracking is now active


WHAT GETS TRACKED:
──────────────────

✅ Page views
✅ User sessions
✅ Traffic sources
✅ User demographics
✅ Real-time visitors
✅ Bounce rate
✅ Time on site

💡 IMPORTANT TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. BACKUP REGULARLY:
   ⚠️ All data is in browser localStorage
   ⚠️ Can be lost if browser data is cleared
   ✅ Export backups weekly
   ✅ Save backup files safely

2. ADSENSE APPROVAL:
   ⚠️ Your site needs AdSense approval first
   ⚠️ Add site to AdSense account
   ⚠️ Wait for approval (can take days/weeks)
   ✅ Once approved, ads will show

3. ANALYTICS VERIFICATION:
   ✅ After adding Analytics ID, wait 24 hours
   ✅ Check Analytics dashboard for data
   ✅ Real-time reports show immediate data

4. MULTIPLE AD UNITS:
   ✅ You can create different ad units
   ✅ Test which performs best
   ✅ Update code anytime in settings


🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: Ads not showing?
A: 
   - Check AdSense approval status
   - Wait 30 minutes after adding code
   - Clear browser cache
   - Check browser console for errors

Q: Analytics not tracking?
A: 
   - Verify Measurement ID is correct
   - Wait 24-48 hours for data
   - Check real-time reports first
   - Ensure ID starts with G- or UA-

Q: Settings not saving?
A: 
   - Check browser localStorage is enabled
   - Try different browser
   - Disable browser extensions
   - Clear cache and try again

Q: Lost all data?
A: 
   - Restore from backup files
   - Check if localStorage was cleared
   - Reconfigure from settings panel
