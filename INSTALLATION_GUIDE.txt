---------------------------------
SOLUTION:
✓ Check password (copy-paste to avoid typos)
✓ Use 123456 as 2FA code for demo
✓ Check if IP is blocked (wait 24 hours)
✓ Try different browser
✓ Clear browser cache


PROBLEM: 5 failed attempts - BLOCKED
------------------------------------
SOLUTION:
✓ Wait 24 hours for auto-unblock
✓ Or use different device/network
✓ Or clear localStorage (loses data):

---------------------------
SOLUTION:
✓ Enable JavaScript in browser
