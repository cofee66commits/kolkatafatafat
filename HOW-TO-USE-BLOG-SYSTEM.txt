🎯 OVERVIEW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

This is a STATIC WEBSITE blog system that works without a database!
- Posts are stored in browser localStorage
- Generate HTML files for each post
- Upload to Cloudflare Pages / AWS S3 / any static hosting
- Perfect for static hosting with zero backend requirements


📂 IMPORTANT FILES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. admin-posts.html     → Admin panel to manage posts (NO LOGIN REQUIRED)
2. posts-manager.js     → Core blog system (handles all post operations)
3. news.html           → News category page (displays all news posts)
4. mall.html           → Mall category page (displays all mall posts)
5. index.html          → Homepage (shows recent posts from both categories)


🚀 HOW TO USE - STEP BY STEP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: ACCESS ADMIN PANEL
───────────────────────────
Open: admin-posts.html in your browser
✅ No username/password required
✅ Works completely offline


STEP 2: ADD A NEW POST
───────────────────────
1. Choose Category: News or Mall
2. Fill in the form:
   - Post Title (required)
   - URL Slug (optional - auto-generated)
   - Featured Image URL (optional)
   - Excerpt (required - short description)
   - Full Content (required - HTML supported)
   - Meta Description & Keywords (optional - for SEO)

3. Click "Add Post" button

4. SUCCESS! You'll see:
   - ✅ Confirmation message
   - 📥 "Download HTML File" button appears


STEP 3: DOWNLOAD THE HTML FILE
───────────────────────────────
Click the "📥 Download HTML File" button

You'll get a file like: "my-post-title.html"

This is a complete, ready-to-upload HTML page!


STEP 4: UPLOAD TO YOUR HOSTING
───────────────────────────────

🔹 FOR CLOUDFLARE PAGES:
   1. Create folders: /news/ and /mall/
   2. Upload post HTML files to respective folders
   3. Example: Upload "my-news.html" to /news/my-news.html

🔹 FOR AWS S3:
   1. Create folders: news/ and mall/
   2. Upload HTML files
   3. Set permissions: public-read
   4. Enable Static Website Hosting

🔹 FOR ANY HOSTING:
   - Upload to: yourdomain.com/news/post-name.html
   - Upload to: yourdomain.com/mall/post-name.html

💾 BACKUP YOUR DATA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Posts are stored in browser localStorage!

TO BACKUP:
1. Open browser console (F12)
2. Type: localStorage.getItem('blogPosts')
3. Copy the output
4. Save to a text file

TO RESTORE:
1. Open browser console
2. Type: localStorage.setItem('blogPosts', 'PASTE_YOUR_BACKUP_HERE')


🚨 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: Posts not showing on news.html or mall.html?
A: Make sure posts-manager.js is loaded properly

Q: Can't see my posts after closing browser?
A: Posts are saved in localStorage - they persist!
   But if you clear browser data, they'll be lost
   Always backup your posts!

Q: How to add posts from different computer?
A: Download HTML files and upload to hosting
   Or transfer localStorage data

Q: Posts showing on admin but not on live site?
A: Admin shows posts from localStorage
   Live site needs HTML files uploaded to server
