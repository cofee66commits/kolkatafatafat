   📝 MALL POSTS - INDEX.HTML में कैसे ADD करें?
═══════════════════════════════════════════════════════════════════

Index.html में "🛍️ Mall Today - Latest Updates" section है
जो automatically last 3 mall posts display करता है।

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ METHOD 1: ADMIN PANEL से ADD करें (EASIEST)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Open Admin Panel
→ Double-click: admin-posts.html

STEP 2: Select "Mall Info" Tab
→ Click on "🏬 Mall Info" tab

STEP 3: Fill Details
→ Title: Mall का नाम/offer
   Example: "South City Mall - Diwali Sale 2025"

→ Content: पूरी information (HTML supported!)
   Example:
   ```
   <h3>South City Mall Diwali Sale</h3>
   <p>Date: 28 Oct - 5 Nov 2025</p>
   <p>Discount: Up to 70% OFF</p>
   <ul>
     <li>Fashion brands 50% OFF</li>
     <li>Electronics 40% OFF</li>
     <li>Food court special offers</li>
   </ul>
   <p><strong>Timing:</strong> 10 AM - 10 PM</p>
   ```

STEP 4: Publish
→ Click: "Publish Post" button

STEP 5: Save HTML File
→ Browser will download: the-post-title.html
→ Upload करें: S3/Cloudflare (same location as index.html)

✅ DONE! Post automatically show होगा:
   - index.html (last 3 posts)
   - mall.html (all posts)

🔧 METHOD 2: INDEX.HTML को DIRECTLY MODIFY करें
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚙️ METHOD 3: SETTINGS PAGE से MANAGE करें
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Open Settings
→ Double-click: settings.html

STEP 2: Go to "Blog Posts Management"
→ Scroll down to "Blog Posts Management" section

STEP 3: View/Delete Posts
→ आप यहां से:
   ✓ सभी mall posts देख सकते हैं
   ✓ Delete कर सकते हैं
   ✓ Export कर सकते हैं (backup)
   ✓ Import कर सकते हैं (restore)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 HOW IT WORKS (TECHNICAL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Posts stored in: localStorage (browser memory)
2. Category: "mall" or "news"
3. JavaScript function: displayPostsList('mall', 'recentMallHome', 3)
   - 'mall' = category
   - 'recentMallHome' = div ID जहां show होगा
   - 3 = कितने posts show करने हैं

CODE LOCATION (index.html, line 150-153):
```javascript
document.addEventListener('DOMContentLoaded', function() {
    displayPostsList('news', 'recentNewsHome', 3);
    displayPostsList('mall', 'recentMallHome', 3);
});
```

अगर आप 5 posts show करना चाहते हैं:
```javascript
displayPostsList('mall', 'recentMallHome', 5);
                                           ↑
                                    change 3 to 5
```
