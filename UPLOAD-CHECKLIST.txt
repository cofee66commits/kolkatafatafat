🔧 AWS S3 SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Create Bucket
1. AWS Console → S3
2. Create bucket: "mallkolkata"
3. Region: ap-south-1 (Mumbai)
4. Uncheck "Block all public access"
5. Create bucket

STEP 2: Enable Static Website
1. Bucket → Properties
2. Static website hosting → Enable
3. Index document: index.html
4. Save changes

STEP 3: Set Bucket Policy
1. Bucket → Permissions
2. Bucket policy → Edit
3. Paste this:

{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicReadGetObject",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::mallkolkata/*"
    }
  ]
}

4. Save changes

STEP 4: Upload Files
1. Bucket → Upload
2. Add files (सभी HTML, CSS, JS files)
3. Permissions → Grant public-read
4. Upload

STEP 5: Test
1. Copy bucket website URL:
   http://mallkolkata.s3-website.ap-south-1.amazonaws.com
2. Open in browser
3. Check console (F12)
4. Should show: "✅ Found txt file: 2025-10-28.txt"
