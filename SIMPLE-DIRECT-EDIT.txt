Q: Editable table नहीं दिख रहा?
A: → Refresh page (F5)
   → Check: index.html latest version?
   → Browser console check करें (F12)

Q: Sum auto-calculate नहीं हो रहा?
A: → Exactly 3 digits type करें (239, not 23)
   → Only numbers (0-9)
   → Check console for errors

Q: Save नहीं हो रहा?
A: → Check: कम से कम 1 round filled?
   → Valid 3 digits?
   → localStorage enabled?

Q: Saved results नीचे नहीं दिख रहे?
A: → Scroll down करें
   → "RESULT TODAY (Saved)" section देखें
   → Refresh page
