1. In your Cloudflare Pages project, create folders:
   /news/
   /mall/

2. Upload the HTML file:
   - News post → Upload to /news/ folder
   - Mall post → Upload to /mall/ folder

3. Done! Visit:
   yoursite.com/news/my-post-name.html

🌐 FOR AWS S3:

1. In your S3 bucket, create folders:
   news/
   mall/

2. Upload HTML file to appropriate folder

3. Set permission: public-read

4. Done! Post is live!
